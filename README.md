# WarmPink
Feminine looking theme for Xfwm4, GTK 2.0, 3.0, 3.2 and Cinnamon.

# Installation:
Run `git clone https://github.com/usr-mary/WarmPink.git`

Copy or move the folder to `~/.themes/` or `/usr/share/themes/` (create it if necessary).
